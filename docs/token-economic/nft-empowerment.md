# NFT Empowerment

NFT transforms into different forms in different scenes and games, and interacts with the game through skill attributes, so that a single NFT is not just an unmodifiable picture, it will interact within different ecologies of NnsDAO.

NFT is a non-convertible identity symbol within the reputation system, it is not tradable, but can be reputation collateralized. The only way it can be acquired is through "participation contributions", which grow through continuous accumulation by individuals and gradually evolve into NFT collections.
